# InsightStay Frontend

## Overview

This is the frontend application for the InsightStay platform. It is built using React, Vite, and Tailwind CSS and provides the user interface for accessing guest feedback insights and platform features.

## Features

- Responsive Navigation Bar
- Hero Section
- Reusable Card Components
- Footer Component
- Home Page
- About Page
- Dashboard Page
- Login Page
- React Router Navigation

## Technologies Used

- React.js
- React Router DOM
- Tailwind CSS
- Vite
